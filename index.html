About me

First and foremost, I am a C# programmer with in-depth knowledge using the Unity Game Engine, and I have a passion for programming AI and VR experiences. I consult on each stage of the software development life cycle including level design, implementation and testing phases. I can draw experience from developing my own personal projects that I completed during Game Jams and University.


When I first picked up a Super Nintendo controller, I fell in love with video games, and ever since then I've had a natural ability to play-test all genre's of games on any console or platform. I think it is very important to be a good communicator to find solutions with the development team and also thinking 'outside the box' to find the bugs which may be overlooked in a typical user journey.


I started my programming quest by teaching myself front-end web development with HTML, CSS and Javascript, so I am able to work on websites too, but my favourite programming language is C#.
